<h1 align='center'>hard-ship-view</h1>

项目于2020.5.13建立，自己搭建自己组件库，有兴趣的可以加入我。

## 特性
- :dvd:**组件库替换**：搭建自己喜欢的组件使用自己组件库。

## 项目启动

```sh
// 下载项目单独运行
git clone https://github.com/ludan1107743896/HardShipView.git
npm install 
npm run start

// 运行 stroyBook 
npm run stroybook 

```
## 目录结构
```
- src
  |- components  //项目自定义组件
  |- styles   //全局配置样式
    |- _animation // 引用全局样式函数（动画函数）
    |- _mixin // 定义样式函数
    |- _reboot // 全局基本样式
    |- _variables // 基本变量模块样式
  |- App //组件入口
```
